# Development Protocol (Reymer Skills Model)

We follow the "skills-first" architecture.

Concept mapping:

Reymer lesson term → Project meaning

Skill
= atomic capability with contract (SKILL.md)

Capability
= domain (career_os)

Agent
= orchestrator + skills execution

MCP (future)
= external tool access layer

Rule:
Implement behavior as skills, not scripts.
Skills must be composable.
