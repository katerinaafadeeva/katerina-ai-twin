# Product Vision — Katerina AI Twin OS

## What we are building
A personal AI Twin operating system.

Not a Telegram bot.
Not a job scraper.

A decision automation system that gradually replaces routine cognitive work.

Initial domain: Career Management.

Future domains:
- communication automation
- planning & calendar
- learning optimization
- financial routines

## Core idea
User becomes operator of an AI team.
AI agents execute workflows continuously.

## Interaction planes
Control Plane:
Telegram — fast commands, approvals, notifications.

Workspace Plane:
Web UI — analytics, strategy, review.

Data Plane:
External sources:
- HH.ru (primary funnel)
- Telegram channels
- future integrations

## North Star
User stops browsing jobs manually.
System surfaces only high-value decisions.
