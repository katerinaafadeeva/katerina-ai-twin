# Vacancy Ingest — Telegram (v0.1)

## Когда активируется
- Пользователь пересылает сообщение в Telegram-бот
- Пользователь просит: "добавь эту вакансию", "сохрани", "разбери пост"

## Что делает
1) Сохраняет сырой текст поста + метаданные (дата, источник/канал, ссылка, message_id если есть)
2) Определяет: это похоже на вакансию или нет (простая эвристика v0.1)
3) Создаёт запись `job_raw` и событие `vacancy.ingested`

## Выход
- Подтверждение пользователю (RU)
- `job_raw` сохранён в SQLite
- Событие для дальнейших шагов: скоринг/политика

## Ограничения v0.1
- Без чтения каналов напрямую (пока только форварды)
- Без автопарсинга сложных вложений
